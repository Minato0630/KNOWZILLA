Introduction to HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages and web applications. With Cascading Style Sheets (CSS) and JavaScript, it forms a triad of cornerstone technologies for the World Wide Web.

HTML was created by Tim Berners-Lee in late 1991 but "HTML 2.0" was the first standard HTML specification which was published in 1995. HTML is a simple and flexible language that is easy to learn and use.


Why Learn HTML?

  • HTML is the foundation of all web pages. Without HTML, you wouldn't be able to create web pages.
  • It is easy to learn and use.
  • HTML is supported by all browsers.
  • It is free and open-source.
  • HTML is used to structure content on the web.
  • It is the first step in learning web development.
  • HTML is essential for creating accessible web content.

HTML Basics

Basic HTML Structure 


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="utf-8"/>
    <meta content="width=device-width, initial-scale=1.0" name="viewport"/>
    <title>Document</title>
</head>
<body>
    <h1>Hello, World!</h1>
</body>
</html>

Simple Paragraph Example 


<!DOCTYPE html>
<html lang="en">

Simple List Example 


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="utf-8"/>

        <li>Item 1</li>
        <li>Item 2</li>

    </ul>
</body>
</html>

HTML Features

Semantic Elements

HTML5 introduced semantic elements that clearly describe their meaning in a human- and machine-readable way.

Multimedia Support

HTML supports multimedia elements like audio and video, making it easier to embed media content.

Forms and Input

HTML provides a wide range of form elements and input types to create interactive forms.

Accessibility

HTML includes features that help create accessible web content for users with disabilities.
