Simple Addition Example 

               
               #include <stdio.h>
               
               int main() {
                   int num1 = 5;
                   int num2 = 10;
                   int sum = num1 + num2;
                   printf("The sum is: %d", sum);
                   return 0;
               }

Simple Loop Example 

               
               #include <stdio.h>
               
               int main() {
                   for (int i = 1; i <= 5; i++) {
                       printf("Count: %d\n", i);
                   }
                   return 0;
               }

C Features

Procedural Language

C is a procedural programming language, which means it follows a sequence of steps to achieve a desired outcome.

Low-Level Access

C provides low-level access to memory and system resources, making it ideal for system programming.

Portability

C code is highly portable and can be run on various platforms with minimal changes.

Efficiency

C is known for its performance and efficiency, making it suitable for high-performance applications.
