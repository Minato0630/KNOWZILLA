Introduction to Java

Java is a high-level, class-based, object-oriented programming language that is designed to have as few implementation dependencies as possible. It is a general-purpose programming language intended to let application developers write once, run anywhere (WORA), meaning that compiled Java code can run on all platforms that support Java without the need for recompilation.

Java was developed by James Gosling at Sun Microsystems and released in 1995. It has since become one of the most widely used programming languages, particularly for client-server web applications. Java's syntax is similar to C++, but it is designed to be simpler to use and enforce an object-oriented programming model.


Why Learn Java?

  • Java is one of the most popular programming languages in the world.
  • It is used for developing mobile apps, web apps, desktop apps, games, and much more.
  • Java has a rich API and a vast ecosystem of libraries and frameworks.
  • It is platform-independent at both the source and binary levels.
  • Java has strong memory management and security features.
  • Java is known for its performance, scalability, and reliability.
  • It has a large and active community, which means plenty of resources and support are available.

Java Basics

Hello World Example 

               
               public class HelloWorld {
                   public static void main(String[] args) {

                   }
               }

Simple Addition Example 

               
               public class Addition {
                   public static void main(String[] args) {
                       int num1 = 5;
                       int num2 = 10;
                       int sum = num1 + num2;
                       System.out.println("The sum is: " + sum);
                   }
               }

Simple Loop Example 

               
               public class LoopExample {
                   public static void main(String[] args) {
                       for (int i = 1; i <= 5; i++) {
                           System.out.println("Count: " + i);
                       }
                   }
               }

Java Features

Object-Oriented

Java is an object-oriented programming language. Everything in Java is an object which makes it easy to extend.

Platform-Independent

Java code is compiled into bytecode which can run on any platform using the Java Virtual Machine (JVM).

Simple and Secure

Java is designed to be easy to use and secure, with features like garbage collection and exception handling.

Rich API

Java provides a rich set of APIs for various tasks, including networking, data structures, and graphical user interfaces.
